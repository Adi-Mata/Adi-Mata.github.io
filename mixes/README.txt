Drop your mix files in this folder.

The browser will try WAV first, then fall back to MP3.
You can provide either format (or both).

Expected file names (from index.html):
- dark-corridors-vol1.mp3  (or .wav)
- rollers-and-tearouts.mp3 (or .wav)
- black-sun-sessions.mp3   (or .wav)

Or rename them in index.html to match your actual file names.

Tip: For streaming, MP3 at 320kbps is recommended.
A 20-min mix will be ~45MB as MP3 vs ~200MB as WAV.
